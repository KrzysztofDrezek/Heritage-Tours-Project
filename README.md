# Heritage Tours – Full Stack Web Application

## 📌 Overview
Heritage Tours is a full-stack web application designed to manage and book guided tours.  
Users can browse tours, check availability, and make bookings through a simple and user-friendly interface.

## 🚀 Features
- Browse available tours
- Search by city and theme
- Check availability
- Booking system with login
- Session-based authentication
- Dynamic availability updates
- View bookings

## 🛠️ Technologies
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: SQLite
- Other: REST API, Express-session

## 🧠 What I Learned
- Building REST APIs with Express
- Connecting frontend with backend
- Managing authentication and sessions
- Working with relational databases
- Handling dynamic data updates
